Spring Cloud Auth 【认证授权】

发表于 2019-10-23 更新于 2019-10-24 阅读次数：

在微服务中认证中，认证和授权是最主要的功能，认证【Authentication】就是在通过用户提交的信息，确认是否是用户本人，比如账号密码登录，手机验证登录，扫码登录，第三方登录。授权【Authorization】就是在用户登录以后，可以看到的数据，有菜单，页面， 按钮，接口，同时还要考虑到不同的租户，不同的组织下大权限问题，还有同一个接口所能看到的数据权限问题。我讲原理，再实践。如我有错，可以在评论区留言，欢迎指正。
